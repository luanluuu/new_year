# 除夕快乐分享项目

这是一个简单的除夕快乐网页，包含倒计时、红灯笼和烟花效果。

## 如何预览

在本地打开 `index.html` 即可预览效果。

## 如何分享给朋友（无需云服务器）

由于你没有云服务器，推荐使用以下免费的静态托管服务：

### 方案一：GitHub Pages（推荐）

1.  注册一个 [GitHub](https://github.com/) 账号。
2.  创建一个新的仓库（Repository），例如命名为 `chuxi-happy`。
3.  将本项目的所有文件（`index.html`, `style.css`, `script.js`）上传到该仓库。
4.  进入仓库的 **Settings** -> **Pages**。
5.  在 **Source** 下选择 `main` 分支，点击 **Save**。
6.  等待几分钟，GitHub 会生成一个链接（例如 `https://yourname.github.io/chuxi-happy/`），你可以把这个链接分享给朋友。

### 方案二：Vercel（速度快）

1.  注册 [Vercel](https://vercel.com/) 账号（可以使用 GitHub 账号登录）。
2.  点击 **Add New...** -> **Project**。
3.  导入你刚才创建的 GitHub 仓库。
4.  点击 **Deploy**。
5.  Vercel 会自动生成一个类似 `chuxi-happy.vercel.app` 的链接，速度通常比 GitHub Pages 快。

### 方案三：Gitee Pages（国内访问快）

1.  注册 [Gitee](https://gitee.com/) 账号。
2.  创建仓库并上传代码。
3.  进入 **服务** -> **Gitee Pages** 开启服务。
    *注意：Gitee Pages 目前可能需要实名认证，且有时会暂停服务，建议优先考虑前两种方案。*
